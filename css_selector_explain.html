CSS Selectors

Name

Element

Definition
Class Selector .hat The Class Selector finds all the elements that have the class of hat.
Element Selector p The Element Selector will select all the tags that have the p tag.
Id Selector #name The Id Selector will select the element with the id of name.
Element Class Selector p.hat The Element Class Selector will select only the p tags with the class of hat.
Universal Selector * The Universal Selector will select every element.
Group Selectors h1, h2, p The Group Selector will selects all the h1, h2, and p tags.
