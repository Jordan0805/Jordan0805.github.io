General HTML Tags

Name

Element

Definition
Declaration !DOCTYPE All HTML documents must start with !DOCTYPE Declaration. The !DOCTYPE tells the browser what type of document is going to be displayed.
Html html The html tag represents the root of an HTML document. The html tag is the container for all other HTML elements (except for the !DOCTYPE tag).
Body body The body tag defines the document's body. The body element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
Head head The head element is a container for metadata and is placed between the HTML tag and the body tag.
  • The following elements ca go inside the head element:
  • title
  • style
  • base
  • link
  • meta
  • script
  • noscript
Title title The title tag defines the defines the document. The title must be text-only, and it is shown in the browser's search engine algorithms to decide the order when listing pages in search results. The title tag is required in HTML documents!
  • The title element:
  • defines a title for the toolbar
  • provides a title for the page when it is added to favorites
  • displays a title for the page in search-engine results
Headings h1, h2, h3, h4, h5, h6 h1 defines the most important heading. h6 defines the least important heading.
Paragraph p The p tag defines a paragraph
Links a | Attribute: href The tag defines a hyperlink, which is used to link one page to another.
Images img | Attributes: src, alt, width, height
  • The img tag presents to the page a image that will be shown.
  • The attribute src is the specific path to the image.
  • The alt attribute is what will be shown if the src is incorrect or there is no image to be shown.
  • To size the image you would use the width and height attributes.
Break Rule br The br tag inserts a single line break.
Horizontal Rule hr The hr tag create a line going from left to right on the page and is used on its own line.
Table table The table tag defines an HTML table.
Table Row tr The tr tag defines a table row.
Table Header th The th tag defines a table header.
Table Data td The td tag defines a table data.
Ordered List ol A ol tag defines a ordered list using numbers to represent each list item.
Unordered List ul A ul tag defines a unordered list using dots to represent each list item.
List Item li A li tag defines a list item.
