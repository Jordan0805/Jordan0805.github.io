HTML Form Tags

Name

Element

Definition
Inupt input | Attributes: type, name, value, id The HTML input element is the most used form element. An input element can be displayed in many ways, depending on the type attribute.
  • type: Attributes
  • text: Displays a single-line text input field
  • radio: Displays a radio button (for selecting one of many choices)
  • checkbox: Displays a checkbox (for selecting zero or more of many choices)
  • submit: Displays a submit button (for submitting the form)
  • button: Displays a clickable button
Label label | Attributes: for The label tag defines a label for a bunch of input type elements
  • label: Attributes
  • for: The for tag should be equal to the id attribute in the input element to bind them together
Select select | Attributes: name, id The select element is most often used in form, to collect user input. It is also used to create a drop-down list.
  • The name attribute is needed to reference the form data after the form is submitted.
  • The id attribute is needed to associate the drop-down list with a label.
Textarea textarea | Attributes: id, name, rows, cols
  • The id attribute is needed to associate the text area with a label.
  • The name attribute is needed to reference the form data after the form is submitted.
  • The size of a text area is specified by the cols and rows attributes (or with CSS).
Button button | Attributes: type The button tag defines a clickable button. Inside a button element you can put text (and tags like i, b, strong, br, img, etc). That is not possible with a button created with the input element!
  • Always specify the type attribute for a button element, to tell browsers what type of button it is.
Feildset fieldset The fieldset tag is used to group related elements tn a form. The fieldset tag draws a box around the related elements.
Legend legend The legend tag defines a caption for the fieldset element.
Datalist datalist | Attributes: id
  • The datalist tag specifies a list of pre-defined options for an input element.
  • The datalist tag is used to provide an "autocomplete" feature for input elements. Users will see a drop-down list of pre-defined options as they input data.
  • The datalist element's id attribute must be equal to the input element's list attribute (this binds them together).
Option option | Attributes: value The option tag defines a option in a select list.
  • value: The option tag can be used without any attributes, but you usually need the value attribute, which indicates what is sent to the server on form submission.
